# Glossário — JavaScript Journey

Glossário oficial da disciplina JavaScript.  
Conceitos, termos, comandos e definições organizados de A → Z.

---

## **C**

### **console**
Ferramenta do navegador ou do Node usada para exibir mensagens durante a execução do programa.

### **console.log()**
Comando utilizado para enviar mensagens ao console.  
Link MDN: https://developer.mozilla.org/pt-BR/docs/Web/API/console/log

---

## **J**

### **JavaScript**
Linguagem de programação usada para criar páginas interativas.

---

## **L**

### **log**
Palavra inglesa que significa **registrar**, **anotar**, **imprimir uma informação**.

---

## **S**

### **string**
Conjunto de caracteres representados entre aspas.

---
